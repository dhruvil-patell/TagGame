# TagGame
This is me creating my first game for hackclub and first time creating a proper repository. It is a game where you go back and forth trying to tag the other player to score points first to 15 wins. 


## instructions
Press 'S' To Start The Game
Click 0 To Read the intsructions of how the game works
Use number keys to change the map 
Yellow box appears when one player hits score of 5 and can be used as a powerup
First one to 15 wins
